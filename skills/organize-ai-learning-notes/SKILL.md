# 整理 AI 学习笔记

## 概述

这个 skill 用于把零散、重复、不完整的学习记录整理成可复习、可积累、可继续行动的笔记。重点不是把文字改漂亮，而是提炼真正学到了什么、还没搞懂什么、下一步该做什么。

## 快速开始

1. 先阅读原始学习记录。
2. 区分事实记录、个人理解和未确认内容。
3. 合并重复信息，保留关键术语。
4. 提炼核心收获和未解决问题。
5. 输出一份结构化复盘。
6. 必要时建议拆分为概念笔记或项目笔记。

## 工作流程

### 1. 判断材料类型

- 判断是课堂笔记、视频笔记、文档摘录、实践记录，还是几种混合。
- 如果内容来源很多，先按主题归组。

### 2. 清理原始记录

- 删除明显重复内容。
- 保留 API 名称、模型名、文件路径、命令等关键信息。
- 不要把不确定内容改写成肯定结论。

### 3. 提炼真正吸收的内容

- 提炼 3 到 5 个核心收获。
- 把“知道了什么”和“理解了什么”分开写。
- 把“还不懂什么”单独列出来。

### 4. 形成可继续使用的输出

- 生成一份学习复盘。
- 如有必要，指出哪些内容适合沉淀为概念卡片。
- 如有必要，指出哪些内容值得变成练习或项目。

## 输出规则

- 不要把整理变成泛泛总结。
- 优先保留原始材料中的关键术语。
- 如果资料不足，只给部分结论并明确说明。
- 输出应服务于后续复习和行动，而不是只看起来工整。

## 典型触发语句

- “把我今天的 AI 学习笔记整理一下”
- “从这堆零散记录里提炼重点、问题和下一步”
- “帮我把这些学习内容整理成复盘”
- “看看哪些内容值得沉淀成长期笔记”
