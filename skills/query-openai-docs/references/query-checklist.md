# 查询清单

在查询 OpenAI 官方文档时，优先按下面顺序执行：

1. 明确问题属于哪个产品或主题
2. 优先使用 OpenAI 官方文档 MCP 搜索相关页面
3. 抓取最相关页面的目标段落
4. 只提炼与问题直接相关的信息
5. 回答时先给结论，再说明依据
6. 如果没有查到明确依据，直接说明未查到

- Responses API
- Chat Completions API
- 结构化输出
- 工具调用
- Realtime API
