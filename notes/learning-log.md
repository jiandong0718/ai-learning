## 已吸收的内容

### 1. skills 是什么

- skills 更像某类任务的作战手册
- skill 本体通常是 `SKILL.md` 加上一些可选资源
- skill 可以规定流程、工具优先级、参考资料和脚本使用方式

### 2. mcp 是什么

- MCP 是模型接外部工具的统一协议
- 它让模型以标准方式调用文档、搜索、时间、记忆、网页抓取等能力
- 配置上通常表现为本地 `stdio` 服务或远程 `url` 服务

### 3. skills 和 rules 的区别

- rules 是全局约束，决定必须怎么做
- skills 是任务增强，决定这类事通常怎么做
- rules 偏“边界”
- skills 偏“方法”

### 4. AI 应用工程师要会什么

- 模型基础
- Prompt 与工作流设计
- 工具接入
- 后端工程
- RAG
- Eval 与可靠性
